## SEVERITY LEVELS

| Level | Description | Response Time | Action Required |
|-------|-------------|---------------|-----------------|
| **CRITICAL** | Immediate shutdown required | < 5 seconds | Stop operation immediately |
| **HIGH** | Severe fault, shutdown imminent | < 30 seconds | Reduce load, prepare shutdown |
| **MEDIUM** | Operating fault, attention needed | < 5 minutes | Monitor closely, schedule repair |
| **LOW** | Minor issue, maintenance recommended | < 1 hour | Plan maintenance window |
| **INFO** | Informational, no action required | N/A | Log for reference |

## USAGE INSTRUCTIONS

### 1. Quick Code Lookup
```
Example: DSE3110-003
- DSE = DEEPSEA controller
- 31 = Series 3100
- 10 = Model variant
- 003 = Specific error number
```

### 2. Category Search
- Use prefix to identify system type
- Reference code ranges for subsystem
- Cross-reference related codes

### 3. Diagnostic Flow
1. **Identify code** from controller display
2. **Check severity** level for urgency
3. **Review possible causes** systematically
4. **Follow solutions** step-by-step
5. **Verify repair** before returning to service
6. **Document** findings and actions
